# 要件定義

## 概要

社内向けの動画バッチ編集ツール。  
役員メッセージなど、複数の人物が個別に撮影した動画に対して、同じ編集テンプレートを適用し、一括で完成動画を生成する。

## ユーザー

- **主なユーザー**: 社内映像担当者・広報担当者
- 操作がシンプルなため、結果的に誰でも使える想定

---

## コアコンセプト

### シーン

タイムラインは複数の**シーン**で構成される。シーンは定義された順に連続して再生される。

**シーンの長さ：**
- シーン内に映像オブジェクトがある場合 → その映像が終わった時点でシーン終了
- シーン内に映像オブジェクトがない場合 → 固定秒数を指定する

### オブジェクト

各シーンには複数のオブジェクトを配置できる。同じ時刻に複数のオブジェクトが存在する場合は重ね合わせて合成される。オブジェクトの時間はシーン先頭からの秒数で指定する。

| 種別 | 説明 |
|------|------|
| **映像** | 動画ファイルを指定した位置・タイミングで再生 |
| **画像** | 画像ファイルを指定した位置・タイミングで表示 |
| **テキスト** | 文字列を指定した位置・タイミングで表示 |

各オブジェクトの属性：

| 属性 | 対象 | 内容 |
|------|------|------|
| ファイル | 映像・画像 | 使用するファイルのパス |
| テキスト内容 | テキスト | 固定文字列、または「可変」指定（ジョブ実行時に入力） |
| 表示位置 | 全 | 画面上の X / Y 座標 |
| サイズ | 全 | 幅・高さ |
| 表示開始時間 | 全 | シーン先頭からの秒数 |
| 表示時間 | 全 | 表示し続ける秒数（0でシーン終端まで） |
| 透明度 | 全 | 0〜100% |
| フォント・文字色・背景色 | テキスト | テキスト装飾 |

### テンプレートとジョブの役割分担

| 設定 | テンプレート | ジョブ（動画ごと） |
|------|------------|-----------------|
| シーン構成・順序 | ○ | |
| オブジェクトの位置・サイズ・タイミング | ○ | |
| 固定の映像・画像ファイル | ○ | |
| 可変テキストの位置・スタイル | ○ | |
| 可変テキストの内容 | | ○ |
| メイン映像ファイル | | ○ |
| メイン映像のトリム量（先頭・末尾のカット秒数） | | ○ |

### テンプレート

シーンのリストと出力設定を記述した **YAMLファイル**。GUIで視覚的に編集し、ファイルとして保存・再利用する。

### ジョブ

テンプレート1つ ＋ 処理対象の動画リスト。各動画エントリに指定する内容：
- メイン映像ファイル
- メイン映像のトリム量（先頭・末尾のカット秒数）
- 可変テキストオブジェクトごとのテキスト内容

## 機能要件

### 1. テンプレートエディタ（GUI）

- シーンの追加・削除・並び替えができる
- シーン内にオブジェクト（映像・画像・テキスト）を追加・編集・削除できる
- 各オブジェクトの属性（位置・サイズ・タイミング・装飾など）をGUIで設定できる
- 編集結果をYAMLファイルとして保存・読み込みできる

### 2. ジョブ管理

- ジョブの作成・編集・保存・読み込み・削除ができる
- ジョブにはテンプレートファイルを1つ紐付ける
- ジョブ内に複数の動画エントリを追加できる
- 各動画エントリに対して、メイン映像ファイル・トリム量・可変テキスト内容を設定できる
- 出力フォルダをジョブごとに指定できる

### 3. バッチ実行

- ジョブ内の全動画を順次処理する
- 処理進捗を表示する（何本中何本完了か）
- エラーが発生した時点で処理を中断し、エラー内容を表示する

### 4. プレビュー

- テンプレートを特定の動画に適用した結果を再生プレビューできる（1動画分）
- FFmpegによる事前レンダリングなしに、GUI上でリアルタイムに合成・再生する

### 5. 動作環境

- デスクトップアプリ（Mac / Windows）
