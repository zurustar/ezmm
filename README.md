# ezmm

動画編集デスクトップアプリ。シーンにオブジェクト（映像・画像・テキスト・音声）を重ねて配置し、FFmpeg で動画に書き出す。プロジェクトは YAML ファイルとして保存・読み込みできる。

## できること

- **シーン構成**: 複数のシーンをつなげてタイムラインを組む
- **オブジェクト配置**: 映像・画像・テキスト・音声を自由な位置・タイミングで重ねる
- **プレビュー再生**: 編集内容をリアルタイムでプレビュー
- **動画書き出し**: H.264 / H.265 / VP9、MP4 / MOV / WebM に対応
- **プロジェクト保存**: YAML 形式で保存・読み込み

## 使い方

1. **新規** でプロジェクトを作成する
2. タイムラインの **＋シーン追加** でシーンを追加する
3. シーンブロック内のアイコンでオブジェクトを追加する
   - 🎬 映像 / 🖼 画像 / 📝 テキスト / ♪ 音声
4. 右側のプロパティパネルでファイル・位置・サイズ・タイミングなどを設定する
5. ツールバーの **出力先** フォルダを指定し、**書き出し** ボタンで動画を生成する
6. **保存**（Cmd+S）でプロジェクトを YAML ファイルとして保存する

## キーボードショートカット

| キー | 操作 |
|------|------|
| Space | 再生 / 一時停止 |
| Escape | 停止して先頭に戻す |
| ← / → | 1 フレーム移動 |
| Cmd+N | 新規 |
| Cmd+O | 開く |
| Cmd+S | 保存 |
| Cmd+Shift+S | 名前を付けて保存 |
| Delete / Backspace | 選択中のオブジェクトを削除 |
| Cmd+D | 選択中のオブジェクトを複製 |

## 動作環境

FFmpeg がインストールされており、`PATH` が通っていること。
